# FileSearcher

opis: program przeszukuje zdjęcia na dysku, a poźniej można je wyszukać poprzez ai

użycie:
zainstalowanie requirements.txt
zdefiniowanie lokalizacji folderu, który skrypt ma przeszukać
wpisz w cmd python3 search.py, poczekaj na zakończenie analizacji zdjęć
pisz to co chcesz szukać, have fun!
